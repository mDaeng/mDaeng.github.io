### 라즈베리파이에 mstsc 기능 설치

콘솔창에서 다음 값 입력.
```
pi@raspberrypi ~ $ sudo apt-get update 
pi@raspberrypi ~ $ sudo apt-get install -y xrdp 
pi@raspberrypi ~ $ sudo service xrdp restart 

```
이 후 원도우에서 mstsc로 접속하면 원격접속이 가능하다.
